Java Movie Management System

1. Basic Features

You are expected to write a movie management system in Java. A basic system should have the following features:

  • User and Manager Systems (20%): the program should have different systems for client and mangers. They share the same login window. If you login as a manager, you should be able to add (remove) some movies to (from) the system and see all the movie ratings from different users. The managers should be able to see the statistics of these ratings, movie types and users ages. The statistics should be presented by chart plots and bar plots (visualization required). The user logins to a different window, where they can see the movie information (including images and words). The user can rate and comments about the movie. The ratings and comments must be recorded by the system.
  • Login System(20%). The program should have a login window (Note that a registration system is not necessary. See the advanced features listed below). The profiles (e.g., age, gender, and address) and their account information (e.g., usernames and passwords) must be recorded in a database. Every time a user logins, the program should check whether his or her account exist and if the input password matches records in the dataset.
  • Java GUI (20%). The program should have a Graphical User Interface (GUI). This interface should be user-friendly, with buttons, text fields, and other features to maintain a comfortable experience. The command interface is not allowed.
  • Documentation, Presentation, and Code (20%). 1) Please write a tutorial about the main features in your project, including how do you implement them and how to run your code. Please make sure TAs can run your code on their machine. (10%) 2) Please prepare a presentation for your project by recording a video. The video should record the screen while your code is running on your laptop or desktop (5%). 3) Please prepare your code for submission. The main functions and classes in the code must have comments in a reader-friendly format. (5%)

2. Advanced Features

Note that you can earn 80% of the scores by submitting the project with the above basic features. That is to say, to get a higher score, you may want to consider adding some advanced features: ( Please pick some features from the followings. The corresponding bonus scores will be added to your final score, and the upper bound of your final score is 100%. That's to say, you can get at most 100% by picking four 5% features or one 10% with two 5% features. )

  • Moving recommendation (5%-10%). The users can receive movie recommendations. The recommended movies are among the ones that a user has not rated or watched. You should write an algorithm to determine which movie to recommend. If your algorithm is rule-based (use manually designed logics, e.g., based on features like age and gender), the top-up will be 5%. If the algorithm is based on advanced data mining and machine learning techniques (e.g., collaborative filtering), the top-up will be 10%.
  • Registration system (5%). new users can register in this system. The information must be written to the dataset so that new user can login in.
  • Pictures Uploading (5%). the users can upload pictures when they post their comments to the movie. The managers can upload movie posters when they add movies to the system.
  • Videos Uploading (5%). The managers can upload videos about the movie into the system. The videos can be seen by the users when they review the movie information.
  • Animation (5%). the statistic plots, login window, and registration window should have some background animations so that the interface looks fancier.
  • Web Application (10%-15%). transfer the local system to a web application with basic web tools or more updated tools. These tools include Java servlet (10%), JSP (10%), Struts (10%), Spring (10%) or SpringBoot (15%).
  • Other cool features (5%): if you have other cool ideas for improving your project, please let me know as early as possible (Before Feb. 28th, 2023) and I will consider some extra top-ups.
