CSC-1004: Computational Laboratory Using Java

1. Course Introduction

This course is a computational lab course for the purpose of strengthening programming skill. As a laboratory course, CSC-1004 will be delivered in the format of finishing projects. Self-teaching is especially important for succeeding in this course. Each student is expected to individually implement:

  • A medium-sized java project (2000-5000 lines of code).
  • A smaller python project (around 500 lines of code).

Each student will choose a project from a list of topics (See the Section Project Topics below), and submit the executable program, source code, and necessary documents at the semester end.

Scoring: 1) Java Project (70%), 2) Python Project (20%) and 3) Weekly-report (10%). For the detailed scoring scheme, please check the project introduction below.

2. Project Topics for Java

  • Java Movie Management System. You are expected to write a movie management system in Java. The basic features include 1) User and Manager Systems, 2) Login System, and 3) Java GUI. For more details, please refer to this introduction .

5. Important Notes

  • Plagiarism is strictly prohibited. If the instructor or TAs catch the sign of plagiarism in your project (including copying the reference code that I give you), you will fail this course and take the consequences, including a warning, suspension of your study or even dismissal from SDS. You must finish the coding by yourself or with group members. DO NOT let others borrow your codes for reference. If any plagiarism happens, both the borrower and the lender receive a penalty.
  • Late Policy. A late submission should receive a penalty. The scale of the penalty will be determined by me and the TAs. If you need special care (e.g., for surgery and other health problem), DO NOT wait until the last moment, and please let me know in advance.
